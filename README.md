# Pretzel Pizza

A static website for Pretzel Pizza, featuring artisanal pretzel crust pizzas. Built with HTML, Tailwind CSS, and exported from Google Stitch.

## Features

- Responsive design
- Functional navigation between pages
- Centralized styles in `styles.css`
- Clean project structure
- Compatible with GitHub Pages

## Pages

- `index.html`: Home page with hero, menu highlights, and CTA
- `menu.html`: Full menu page
- `build_your_own.html`: Custom pizza builder

## Setup

1. Clone or download the repository
2. Open `index.html` in a browser
3. For GitHub Pages, upload the files to your repository and enable Pages

## Assets

Place images in the `assets/` folder. The following images are referenced:

- `assets/hero-pizza.jpg`
- `assets/the-salted-pep.jpg`
- `assets/garlic-knot-twist.jpg`
- `assets/signature-pretzel.jpg`
- `assets/custom-pizza-preview.jpg`

Download the original images from the Google links and save them locally.

## Development

- Styles are centralized in `styles.css`
- Tailwind config is in `config.js`
- Fonts and icons are loaded from Google Fonts and Material Symbols
